# Personal Portfolio Website (HTML & CSS)

## Project Overview

This project is a personal portfolio website developed using **HTML5 and external CSS**.  
It showcases personal information, educational background, technical skills, and a contact form in a clean, visually appealing, and responsive layout.

The project demonstrates the practical use of HTML for structure and CSS for styling, layout, and responsiveness.

---

## Objectives

- To design a structured portfolio using semantic HTML5
- To apply external CSS for styling and layout
- To implement hover effects and responsive design
- To understand CSS selectors, box model, and media queries
- To follow proper project and GitHub file structure

---

## Technologies Used

- HTML5  
- CSS3
